Library Management System

The Library Management System is a simple and efficient web application designed to manage library operations. It allows users to add, view, update, and delete books, making library management faster and more organized.

Key Features:

Book Management: Add, edit, and remove books with details like title, author, category, and availability.

Search Functionality: Search for books by title, author, or category.

User-Friendly Interface: Simple and intuitive UI for easy navigation.

Backend Management: Built using Spring Boot and MongoDB for robust and scalable data storage.

RESTful APIs: Enables interaction with the database efficiently using standard HTTP methods (GET, POST, PUT, DELETE).

Technologies Used:

Backend: Spring Boot

Database: MongoDB

Build Tool: Maven

Development Environment: IntelliJ IDEA

Programming Language: Java

Purpose:
This project is designed for beginners to learn full-stack development concepts, understand REST APIs, and practice CRUD operations with a NoSQL database like MongoDB.
